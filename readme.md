> Isomorphic/Univeral WHATWG URL API with some support legacy node URL API


## Install

```
$ npm install iso-url
```


## Usage

```js
const isoUrl = require('iso-url');

isoUrl('unicorns');
//=> 'unicorns & rainbows'
```
